# EAGLE DECO - Site Web

Site web professionnel pour EAGLE DECO, entreprise de peinture et décoration à Abidjan, Côte d'Ivoire.

## 🚀 Déploiement sur Vercel

### Méthode 1: Déploiement automatique
1. Double-cliquez sur `deploy.bat`
2. Suivez les instructions à l'écran
3. Connectez-vous à votre compte Vercel
4. Le site sera déployé automatiquement

### Méthode 2: Déploiement manuel
```bash
# Installer Vercel CLI
npm install -g vercel

# Se connecter à Vercel
vercel login

# Déployer
vercel --prod
```
